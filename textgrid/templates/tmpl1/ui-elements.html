Buttons

Classes

The HighLand template offers you new button classes that can be used along with the default Bootstrap 3 buttons:

Example

<button type="button" class="btn btn-default">Default</button>
<button type="button" class="btn btn-green">Green</button>
<button type="button" class="btn btn-blue">Blue</button>
<button type="button" class="btn btn-dark">Dark</button>
<button type="button" class="btn btn-red">Red</button>
<button type="button" class="btn btn-turquoise">Turquoise</button>
<button type="button" class="btn btn-amethyst">Amethyst</button>

Sizes

Fancy larger or smaller buttons? Add .btn-lg, .btn-sm, or .btn-xs for additional sizes.

Example

<button type="button" class="btn btn-default btn-lg">Large button</button>
<button type="button" class="btn btn-amethyst btn-lg">Large button</button>

<button type="button" class="btn btn-default">Default button</button>
<button type="button" class="btn btn-amethyst">Default button</button>

<button type="button" class="btn btn-default btn-sm">Small button</button>
<button type="button" class="btn btn-amethyst btn-sm">Small button</button>

<button type="button" class="btn btn-default btn-xs">Extra small button</button>
<button type="button" class="btn btn-amethyst btn-xs">Extra small button</button>

Create block level buttons - those that span the full width of a parent - by adding .btn-block.

Example
<button type="button" class="btn btn-default btn-lg btn-block">Block level button</button>
<button type="button" class="btn btn-green btn-lg btn-block">Block level button</button>

Disable state

Make buttons look unclickable by fading them back 50%.

Button element

Add the disabled attribute to <button> buttons.

Example
<button type="button" class="btn btn-lg btn-red" disabled="disabled">Red button</button>
<button type="button" class="btn btn-lg btn-default" disabled="disabled">Button</button>
Anchor element

Add the .disabled class to <a> buttons.

Example
Red link Link
<a href="#" class="btn btn-red btn-lg disabled">Red link</a>
<a href="#" class="btn btn-default btn-lg disabled">Link</a>

Note the .nav-tabs class requires the .nav base class.

Section Header

Wrap your section header in a <h1 class="animated slideInLeft"><span> ... </span></h1>, then center it by wrapping in a .container and a .section-header div for correct look and block positioning.

Remove .animated slideInLeft if you want to get rid of the animation effect.

Example

Section Header

<div class="section-header">
<div class="container">
<div class="row">
<div class="col-md-12">
<h1><span>Section Header</span></h1>
</div>
</div>
</div>
</div>

Page Tip

Wrap your tip contents in a .container and a .page-tip animated slideInDown to get a neat tip box with a slide down effect.

Again, remove .animated slideInLeft if you don't want things to move.

Example

This tip looks great!

<div class="page-tip">
<div class="container">
<div class="row">
<div class="col-md-12">
<p>This tip looks great!</p>
</div>
</div>
</div>
</div>
